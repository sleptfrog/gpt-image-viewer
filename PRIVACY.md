# Privacy Policy / プライバシーポリシー

Last updated: May 21, 2026

## English

GPT Image Viewer is a Chrome extension for viewing and saving images generated in the ChatGPT web UI together with related metadata.

### Data handled by this extension

GPT Image Viewer may process the following data from ChatGPT pages opened by the user:

- generated images
- user-provided image attachments in conversations
- image URLs
- user inputs associated with images
- prompts
- captions
- conversation IDs
- message IDs
- image IDs
- creation and capture timestamps

GPT Image Viewer does not collect passwords, authentication tokens, cookies, payment information, health information, or location information.

### How the data is used

The data is used only to display image metadata in the extension side panel and to save selected images locally with metadata.

### Data sharing

GPT Image Viewer does not send user inputs, prompts, images, metadata, or conversation data to any external server. The developer does not sell, transfer, or share user data with third parties.

### Storage

Metadata and image URL mappings may be stored temporarily in Chrome extension storage. Saved images and sidecar JSON files are stored locally only when the user chooses to save them. For bulk save, files are written only to the local folder selected by the user through the browser folder picker.

Saved image files and sidecar JSON files may include user inputs, prompts, captions, conversation IDs, message IDs, image IDs, and image URLs. If the user shares those saved files, that metadata may also be shared.

### Limited Use

GPT Image Viewer uses handled data only for its single purpose: viewing and saving ChatGPT generated images with related metadata. The developer does not use or transfer user data for purposes unrelated to this purpose, does not sell user data, and does not use user data for creditworthiness or lending purposes. GPT Image Viewer is intended to comply with the Chrome Web Store User Data Policy, including Limited Use requirements.

### Remote code

GPT Image Viewer does not execute remotely hosted code. Extension code is bundled with the extension package.

### Contact

For questions about this privacy policy, contact: <syuu1111@gmail.com>

## 日本語

GPT Image Viewer は、ChatGPT のWeb UIで生成された画像を、関連するメタデータと一緒に閲覧・保存するためのChrome拡張機能です。

### この拡張機能が扱うデータ

GPT Image Viewer は、ユーザーが開いたChatGPTページから、以下のデータを処理する場合があります。

- 生成画像
- 会話内でユーザーが添付した画像
- 画像URL
- 画像に関連するユーザー入力
- プロンプト
- キャプション
- conversation ID
- message ID
- image ID
- 作成日時および取得日時

GPT Image Viewer は、パスワード、認証トークン、Cookie、支払い情報、健康情報、位置情報を収集しません。

### データの利用目的

これらのデータは、拡張機能のサイドパネルで画像メタデータを表示し、ユーザーが選択した画像をメタデータ付きでローカル保存する目的にのみ使用します。

### データの共有

GPT Image Viewer は、ユーザー入力、プロンプト、画像、メタデータ、会話データを外部サーバーへ送信しません。開発者はユーザーデータを第三者へ販売、転送、共有しません。

### 保存

メタデータと画像URLの対応情報は、Chrome拡張機能のストレージに一時的に保存される場合があります。保存された画像とsidecar JSONファイルは、ユーザーが保存操作を行った場合にのみローカルに保存されます。一括保存では、ブラウザのフォルダ選択でユーザーが選んだローカルフォルダにのみファイルを書き込みます。

保存された画像ファイルやsidecar JSONファイルには、ユーザー入力、プロンプト、キャプション、conversation ID、message ID、image ID、画像URLが含まれる場合があります。ユーザーがそれらの保存物を共有した場合、これらのメタデータも共有される可能性があります。

### 限定使用

GPT Image Viewer は、扱うデータを、ChatGPT生成画像と関連メタデータを閲覧・保存するという単一の目的にのみ使用します。開発者は、この目的と関係のない目的でユーザーデータを使用または転送せず、ユーザーデータを販売せず、信用力判断または融資目的でユーザーデータを使用しません。GPT Image Viewer は、Limited Use 要件を含む Chrome Web Store User Data Policy に従うことを意図しています。

### リモートコード

GPT Image Viewer は、リモートでホストされたコードを実行しません。拡張機能で実行されるコードは、拡張機能パッケージ内に含まれています。
